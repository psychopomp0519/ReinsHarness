# Discussion Format Reference

## 3-Round Structure

### Round 1: Individual Analysis
Each agent independently provides:
- Position statement (1-2 sentences)
- Key arguments (3-5 bullet points)
- Supporting evidence (code references, data, examples)
- Confidence level (High / Medium / Low)

### Round 2: Cross-Rebuttal
Each agent reviews others' positions and provides:
- Agreements (what they accept from other agents)
- Challenges (what they dispute, with reasoning)
- Questions (what needs clarification)
- Updated position (if changed based on new arguments)

### Round 3: Consensus
Facilitator (you) synthesizes into:
- Unanimous agreements
- Majority positions (with dissent noted)
- Unresolved tensions (trade-offs the user must decide)
- Final recommendation with confidence level
